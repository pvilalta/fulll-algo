# Algo's workshop

## Requirement

NodeJS

## Installation

execute it at the root directory

```
$ npm install
```

## Workshop #1 : NumberType

run the script with

```
$ node algoNumberType
```

## Workshop #2 : FizzBuzz

run the script with

```
$ node algoFizzBuzz
```

## Test

You can test both script with respectively

```
$ npm test algoNumberType
```

and

```
$ npm test algoFizzBuzz
```

or globally

```
$ npm test
```
